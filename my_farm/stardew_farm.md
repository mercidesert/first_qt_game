# 《星露农场》游戏需求文档

## 一、项目概述

### 1.1 游戏名称
星露农场（Stardew Farm）

### 1.2 游戏类型
农场模拟经营 + 休闲小游戏合集

### 1.3 目标平台
Windows PC（Qt框架）

### 1.4 游戏分辨率
640 x 640 像素（固定窗口）

### 1.5 游戏简介
玩家扮演一位农场主，通过种植作物、钓鱼、战斗、参与节日活动等方式经营自己的农场，与农场猫咪互动，体验休闲的乡村生活。

---

## 二、核心系统

### 2.1 场景系统

游戏包含7个可切换场景：

| 编号 | 场景名称 | 英文名 | 功能描述 |
|------|----------|--------|----------|
| 0 | 主菜单 | MainMenu | 游戏入口，选择进入各游戏模式 |
| 1 | 家园 | Home | 出售作物、睡觉恢复体力、盲盒抽奖 |
| 2 | 节日 | Festival | 与NPC对话、皮埃尔种子商店 |
| 3 | 农场 | Farm | 核心玩法：耕种、收获、猫咪互动 |
| 4 | 钓鱼 | Fishing | 钓鱼小游戏 |
| 5 | 战斗 | Combat | 节奏音游式战斗 |
| 6 | 彩蛋 | Easter | 找彩蛋小游戏 |

### 2.2 全局游戏状态（GameData）

| 属性 | 类型 | 初始值 | 说明 |
|------|------|--------|------|
| gold | int | 150 | 玩家金币 |
| energy | int | 100 | 当前体力 |
| maxEnergy | int | 100 | 最大体力 |
| isInjured | bool | false | 受伤状态（影响战斗/耕种） |
| currentSeason | enum | Winter | 当前季节（冬/春/夏/秋） |
| timeTicks | int | 0 | 季节计时器（200 ticks换季） |
| globalMsg | QString | "" | 全局提示消息 |
| globalMsgTimer | int | 0 | 消息显示计时器 |

### 2.3 季节系统

- **季节顺序**：冬 → 春 → 夏 → 秋 → 冬
- **换季时间**：每 200 ticks（约40秒）
- **季节影响**：
  - 草莓只能在春季种植
  - 向日葵可在夏/秋季种植
  - 跨季作物自动枯萎

## 三、场景详细需求

### 3.1 主菜单（MainMenu）

**界面元素**：
- 标题背景图
- 飞鸟动画（从右向左循环飞行）
- 两个祝尼魔动画（相位相反）
- 5个功能按钮

**按钮功能**：

| 按钮 | 切换场景 | 特殊条件 |
|------|----------|----------|
| Go Home | 家园 | 无 |
| Town Festival | 节日 | 无 |
| Farm | 农场 | 无 |
| River (Fishing) | 钓鱼 | 无 |
| Combat | 战斗 | 受伤时拒绝；体力≥10扣10进入；体力不足提示 |

### 3.2 家园（Home）

**界面布局**：
- 家园背景图
- 出货箱（右侧）- 点击出售
- 床（右上方）- 点击恢复体力
- 电视机（左下方）- 点击触发盲盒

**功能需求**：

| 功能 | 操作 | 效果 |
|------|------|------|
| 出售 | 点击出货箱 | 卖出所有作物和鱼，获得金币 |
| 睡觉 | 点击床 | 体力恢复至最大值 |
| 盲盒 | 点击电视→按Y确认 | 花费100G随机获得1/6诗意文本 |

**盲盒系统**：
- 6种不同的诗意文本
- 显示在信封背景中
- 点击屏幕中心区域关闭

### 3.3 节日（Festival）

**NPC系统**：
- 共22个NPC
- 每个NPC有：名字、点击区域、3段随机对话
- 点击NPC显示对话气泡

**特殊NPC交互**：

| NPC | 触发条件 | 效果 |
|-----|----------|------|
| Harvey | 受伤时点击2次 | 花费50G治疗，解除受伤状态 |
| Abigail | 点击4次 | 跳转到彩蛋场景 |
| Pierre | 点击3次 | 打开种子商店 |

**种子商店**：
- 按1购买：草莓种子（20G）
- 按2购买：向日葵种子（10G）
- 显示商店界面

**表情系统**：
- 点击左上角区域显示随机表情
- 15种不同表情图标
- 表情出现动画（4帧）

### 3.4 农场（Farm）

#### 3.4.1 地图系统
- 地图尺寸：20列 × 10行
- 格子尺寸：32 × 64 像素
- 支持四季背景切换

#### 3.4.2 耕作系统

**操作方式**：
- WASD移动玩家
- 1/2切换手持种子
- J/Space 与面前地块交互

**交互逻辑**（优先级顺序）：

| 情况 | 操作 | 消耗/结果 |
|------|------|----------|
| 未耕地 | 锄地 | 消耗2体力，8%概率挖出种子 |
| 已耕地 + 成熟作物 | 收获 | 草莓→+1收获，变回成熟；向日葵→+1收获+种子，地块清空 |
| 已耕地 + 无作物 | 播种 | 消耗1种子+2体力，需对应季节 |

**作物数据**：

| 作物 | 种子价格 | 售价 | 生长周期 | 可种植季节 |
|------|----------|------|----------|------------|
| 草莓 | 20 G | 20 G | 4阶段（80 ticks） | 春季 |
| 向日葵 | 10 G | 15 G | 4阶段（80 ticks） | 夏季、秋季 |

**作物生长阶段**：种子 → 幼苗 → 成熟 → 结果

#### 3.4.3 体力与受伤限制
- 每次操作消耗2体力
- 体力<2时无法操作
- 受伤状态下无法耕作

#### 3.4.4 HUD显示
- 当前季节
- 手持种子类型
- 草莓/向日葵种子数量
- 草莓/向日葵收获数量
- 季节倒计时（进度条）

### 3.5 钓鱼（Fishing）

**状态机**：

| 状态 | 含义 | 操作 |
|------|------|------|
| 0 | 空闲 | 按空格抛竿（消耗5体力） |
| 1 | 等待 | 自动等待（10-25帧） |
| 2 | 咬钩 | 显示"！"，按空格进入小游戏 |
| 3 | 小游戏 | 追逐鱼，按空格控制绿条 |
| 4 | 结果 | 显示捕获结果 |

**小游戏机制**：
- 空格按住：绿条上升
- 空格松开：绿条下降
- 鱼与绿条重叠 → 捕获进度 +2
- 鱼与绿条分离 → 捕获进度 -1
- 进度≥100 → 成功捕获
- 进度≤0 → 失败

**鱼类系统**（6种）：

| 索引 | 名称 |
|------|------|
| 0 | 河豚 |
| 1 | 金枪鱼 |
| 2 | 河鲈 |
| 3 | 鲇鱼 |
| 4 | 鱿鱼 |
| 5 | 海草 |

**宝藏系统**（50%概率）：

| 奖励类型 | 内容 |
|----------|------|
| 0 | 金币 50-150 |
| 1 | 草莓种子 1-3 |
| 2 | 向日葵种子 1-3 |

**宝箱动画**：5帧开启动画

### 3.6 战斗（Combat）

**核心玩法**：双轨下落式节奏音游

**操作方式**：
- `[F]`键：打击第一轨（黄色判定圈）
- `[空格]`键：打击第二轨（蓝色判定圈）

**判定机制**：

| 判定 | 距离 | 伤害 | 反馈文字 |
|------|------|------|----------|
| PERFECT | ≤30px | 15 | "PERFECT!!" |
| GOOD | ≤65px | 5 | "GOOD!" |
| MISS | 超过判定 | 玩家受伤5 | "MISS!" |

**难度递进**：

| 等级 | 基础速度 | 音符间隔 | 双音符概率上限 |
|------|----------|----------|----------------|
| 1 | 15.5 | 8-32 | 10% |
| 2+ | 12+等级×3.5 | 逐级缩短 | 逐级增加至60% |

**敌人系统**：
- 基础血量：100
- 每级增加：35血量
- 敌人图片尺寸：128×128

**胜负条件**：
- 胜利：敌人HP≤0 → 获得金币奖励（10+等级×10）
- 失败：玩家HP≤0 → 受伤状态（isInjured=true）

**战后选项**：
- 按Y：下一关（保留体力消耗）
- 按N：返回主菜单
- 死亡时空格返回主菜单

### 3.7 彩蛋（Easter）

**游戏规则**：
- 时间限制：75 ticks（约15秒）
- 彩蛋数量：12个
- 随机分布位置（x: 20-600, y: 120-600）

**操作方式**：
- 鼠标点击彩蛋收集
- 彩蛋尺寸：12×12像素

**胜负判定**：
- 胜利：收集全部12个彩蛋 → 显示"EGG HUNTER!"
- 失败：时间归零 → 显示"TIME'S UP!"

**结束返回**：空格键返回节日场景

## 四、猫咪系统（FarmCat）

### 4.1 基础属性

| 属性 | 值 | 说明 |
|------|-----|------|
| 初始位置 | (300, 300) | 屏幕中央 |
| 移动速度 | 1.5 像素/帧 | |
| 活动区域 | 600×600 | 考虑猫咪尺寸的边界 |
| 尺寸 | 48×48 像素 | |

### 4.2 行为状态（共9种）

| state | 动作 | 动画帧数 |
|-------|------|----------|
| 0 | 向下走 | 4 |
| 1 | 向上走 | 4 |
| 2 | 向左走 | 4 |
| 3 | 向右走 | 4 |
| 4 | 躺卧 | 4 |
| 5 | 舔毛 | 4 |
| 6 | 蜷缩 | 2 |
| 7 | 坐（默认） | 4 |
| 8 | 跳跃 | 2 |

### 4.3 行为逻辑
- 每30-150帧随机切换状态
- 状态0-3时移动
- 撞墙后反向反弹
- 动画每6帧切换一次

### 4.4 表情系统
- **触发方式**：点击猫咪
- **表情数量**：15种
- **动画效果**：
  - 4帧出现动画（每3帧切换）
  - 完全显示后停留45帧
- **绘制位置**：猫咪头顶上方

## 五、资源系统

### 5.1 图片资源分类

| 类别 | 数量 | 用途 |
|------|------|------|
| 背景图 | 6张 | 各场景背景 |
| 作物图 | 8张 | 草莓/向日葵4阶段 |
| 玩家动画 | 16张 | 4方向×4帧 |
| 猫咪动画 | 42张 | 9种状态动画 |

| UI元素 | 若干 | 按钮、图标、对话框等 |

### 5.2 资源加载策略
- 使用Qt资源系统（编译时嵌入）
- 单例模式统一管理
- 三种缩放策略：
  - 瓷砖：忽略比例，快速变换
  - 背景：忽略比例，平滑变换
  - UI：保持比例，平滑变换
